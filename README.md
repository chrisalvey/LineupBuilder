# DraftKings Lineup Builder

A web-based tool for building DraftKings Daily Fantasy Sports (DFS) lineups.

## Features

- Upload player CSV files
- Build lineups for different contest types (Classic, Showdown)
- Position-based player selection
- Real-time salary cap tracking
- Easy lineup management

## How to Use

1. Select your contest type (Classic or Showdown)
2. Upload a CSV file with available players
3. Browse players by position using the tabs
4. Click on players to add them to your lineup
5. Monitor your salary cap usage in real-time

## Deployment

This application is hosted on GitHub Pages at: https://chrisalvey.github.io/LineupBuilder/

## Development

This is a simple static web application built with vanilla HTML, CSS, and JavaScript.

### Local Development

Simply open `index.html` in a web browser to run the application locally.

## CSV Format

The application expects a CSV file with the following columns:
- Name
- Position
- Team
- Salary

(Format will be updated based on actual DraftKings CSV structure)
